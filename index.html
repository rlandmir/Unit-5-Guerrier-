Text Formatting

There are three types of text formatting. Those are: bold, italics, and underline.

Inline vs. Block Element

Block element moves the content to a new line. The paragraph and headers elements are block elements. An inline element keeps its content on the same line as the content before and after it. It does not creat a new line. The bold, italics and underline element are inline element
